1.git init 初始化
2.git status 查看文件夹状态
3.git add . 使用git管理文件
4.git commit -m "v1" 生成版本v1
5.git log 查看版本生成日志~~
6.git reflog
7.git reset --hard 版本号  回滚
8.git branch 查看分支
9.git branch 分支名
10. git checkout 分支名
11.git merge 要合并的分支
12.git branch -d 分支名称
13.git remote add 别名 远程仓库地址   起别名
14.git push -u 别名 分支  推送代码
15.git clone 远程仓库地址  拉取代码（默认别名）
16.在公司开发
	git checkout dev  切换到dev分支进行开发
	git merge master  把master分支合并到dev(仅一次）
	git add.
	git commit -m "xx"
	git push origin dev
17.在家中继续写代码
	切换到dev分支进行开发
	git checkout dev
	拉代码
	git pull origin dev
	git add .
	git commit -m "xxx"
	git push origin dev
开发完毕，上线
